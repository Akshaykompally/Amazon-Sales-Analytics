# 📊 Amazon Sales Dashboard

An interactive **Amazon Sales Dashboard** built with **Power BI** to visualize and analyze Amazon sales data. This project provides valuable business insights through dynamic dashboards, helping users understand sales performance, customer behavior, and product trends.

---

## 📖 Project Overview

The **Amazon Sales Dashboard** is a Business Intelligence project developed using **Microsoft Power BI**. It transforms raw Amazon sales data into meaningful visualizations, enabling users to explore key performance metrics and make data-driven decisions.

The dashboard includes interactive charts, KPIs, filters, and reports that provide a comprehensive view of sales across products, categories, regions, and time.

### 1. Clone the Repository

```bash

```

### 2. Open the Project

* Install **Microsoft Power BI Desktop**.
* Open the file:

```text

```

If required, refresh the data using:

